### Sobre mí

Vengo de una base fuerte en **algoritmos y estructuras de datos** —grafos, búsqueda, simulación— y estoy llevando eso hacia el trabajo con **datos**: análisis, consultas SQL y Python. Me interesa modelar problemas y resolverlos con código limpio, y ahora mismo estoy metido en pandas, visualización y mis primeros modelos.

Abierto a colaborar en proyectos de análisis o ciencia de datos.
